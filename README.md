# Fullstack Grind

A deliberate practice repository for becoming a product-ready full-stack engineer.

## Focus
- JavaScript fundamentals
- React and Next.js
- API integration
- Ruby on Rails basics
- Interview preparation
- Real-world product feature building

## Repository Goals
- Build strong frontend fundamentals
- Practice backend integration
- Ship mini-projects weekly
- Document learning clearly
- Prepare for full-stack interviews

## Structure
- `notes/` → concise study notes
- `exercises/` → isolated coding drills
- `mini-projects/` → focused feature builds
- `projects/` → more complete applications
- `progress-log.md` → weekly improvement log

## Current Stack
- JavaScript
- React
- Next.js
- Tailwind CSS
- Node.js
- Ruby on Rails (in progress)
